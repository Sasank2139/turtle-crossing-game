# Turtle Crossing Game 🐢🚗

A Frogger-inspired Turtle Crossing Game built using Python and the Turtle graphics library as part of Angela Yu's **100 Days of Code: The Complete Python Pro Bootcamp**.

## Features

- Player-controlled turtle
- Randomly generated moving cars
- Collision detection
- Level progression
- Increasing game difficulty
- Game Over screen

## Technologies Used

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

## Project Structure

```
turtle-crossing-game/
│── main.py
│── player.py
│── car_manager.py
│── scoreboard.py
```

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run:

```bash
python main.py
```

## Controls

- **Up Arrow (↑)** → Move the turtle forward
